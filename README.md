# Python File Automation Sorter

A Python-based automation tool that organizes files in a directory into categorized folders based on file type. Built using Jupyter Notebook and Python’s built-in libraries to demonstrate file handling, automation, and workflow optimization.

---

## Features

- Automatically scans a target directory
- Creates folders if they do not exist
- Sorts files into categories:
  - CSV files
  - Image files
  - Text files
- Moves files into appropriate folders
- Prevents duplicate file movement

---

## Tools Used

- Python
- Jupyter Notebook
- os module
- shutil module

---

## How It Works

1. Reads all files in a selected directory  
2. Identifies file types using extensions  
3. Creates necessary folders if missing  
4. Moves files into corresponding folders automatically
